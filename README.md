# Blog app

## requirement

- user can post and publish blog content
- user can see post
- authentication system
- user can see their own profile

## Table

- post

  - id
  - title
  - content
  - authorId
  - createdAt
  - updatedAt
  - published

- user

  - id
  - name
  - email
  - password
  - createAt
  - updateAt
  - profile

- profile

  - id
  - bio
  - createAt
  - updateAt
  - userId

## Technology Stack:

- graphql
- typescript
- postgresql
- prisma
